正式な防災判断を代替するものではありません。実際の対応は、ホームのマニュアル、自治体情報、消防・警察・管理者指示を優先してください。

    過去結果

    結果はこの端末内に保存されます。個人情報や回答データは外部送信しません。

    使い方

    1. 対象を「利用者向け」または「職員向け」から選ぶ。
    2. 5問、10問、20問、30問から訓練量を選ぶ。高難度・専門複合問題は、通常・しっかり・ランダム模試の中に混ざります。
    3. 問題文の状況を読み、最も安全な行動を選ぶ。
    4. 正解と理由を確認する。
    5. 訓練後に、分類別正答率と誤答傾向を見る。

    GitHub Pagesで公開する基本手順

    1. GitHubで新規リポジトリを作る。
    2. このフォルダ内のファイルをそのままアップロードする。
    3. Settings → Pages で、公開元を main ブランチの / にする。
    4. 数分後、PagesのURLから開く。

    公開リポジトリに、実在利用者の氏名・住所・個別情報は入れないでください。ホーム固有情報を入れる場合は、非公開運用かローカル版を推奨します。
