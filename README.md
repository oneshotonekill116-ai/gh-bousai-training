# ホーム防犯・防災 判断トレーニング

グループホーム向けの防犯・防災判断トレーニング用PWAです。
GitHub Pagesにそのまま置ける静的HTML/CSS/JavaScript構成です。

## できること

- 利用者向けモード
  - 防犯、地震、火災、洪水時に本人が安全な行動を選ぶ訓練
- 職員向けモード
  - 初動、通報、避難誘導、点呼、記録、管理者連絡の判断訓練
- ランダム出題
  - 利用者向け140問
  - 職員向け140問
- 分類別結果
  - 防犯、地震、火災、洪水の正答率
- 誤答傾向
  - 戻る型、見に行く型、通報迷い型など
- 端末内履歴保存
  - localStorageのみ使用
  - 外部送信なし
- PWA対応
  - スマホのホーム画面に追加可能
  - Service Workerによる簡易オフライン対応

## GitHub Pages公開手順

1. GitHubで新規リポジトリを作成する。
2. このフォルダの中身をリポジトリ直下にアップロードする。
3. `index.html` がリポジトリ直下にあることを確認する。
4. GitHubの `Settings` → `Pages` を開く。
5. `Build and deployment` の `Source` を `Deploy from a branch` にする。
6. Branch を `main`、フォルダを `/ (root)` にする。
7. Save後、表示されるGitHub PagesのURLを開く。

## 注意

このアプリは訓練補助ツールです。
正式な防災判断、消防計画、BCP、警察・消防・自治体の指示を代替するものではありません。

公開リポジトリに、実在利用者の氏名、住所、電話番号、障害特性、個別支援情報を入れないでください。
ホーム固有情報を入れる場合は、非公開リポジトリまたはローカル運用を推奨します。

## 問題の編集

### 利用者向け問題

`data/userQuestions.js` を編集します。

### 職員向け問題

`data/staffQuestions.js` を編集します。

問題データ例：

```javascript
{
  id: "user_crime_001",
  audience: "user",
  category: "防犯",
  situation: "昼間・職員がいない",
  difficulty: 2,
  question: "知らない人が玄関で『中に入れてください』と言っています。どうしますか？",
  choices: ["すぐ開ける", "ドアを開けず、職員に連絡する", "少しだけ中に入れる"],
  answer: 1,
  reason: "知らない人を中に入れない。まず職員か決められた連絡先に確認する。",
  trap: "開けてしまう型",
  skills: ["防犯判断", "相談行動", "衝動抑制"]
}
```

`answer` は `choices` の番号です。0始まりなので、2番目の選択肢が正解なら `1` です。

## 推奨運用

- ホーム会議：10問
- 防災訓練日：20問
- 夜勤前確認：職員向け5〜10問
- 応援職員向け：職員向け20問
- 利用者向け：結果の点数より、間違えた場面を実物確認する

  - 利用者向け：120問 → 140問
  - 職員向け：120問 → 140問
- 難度8〜9の問題を追加しました。
  - 職員不在時に電話がつながらない

- Service Workerをv4に更新し、古い `app.js` がキャッシュから残り続けにくいよう network-first に変更しました。
